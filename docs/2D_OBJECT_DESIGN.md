# 2D表示用オブジェクトデザイン案

## 📋 概要

2Dドットライクな画面に合わせて、各種オブジェクトの表示を調整します。

## 🎨 デザイン方針

### 1. 感情オブジェクトの形状

各感情カテゴリに応じて異なる形状を採用：

| 感情 | 形状 | 色 | アニメーション |
|------|------|-----|---------------|
| **joy** | 星形（⭐） | ゴールド (#ffd700) | キラキラ（回転+パルス） |
| **peace** | 円形（●） | 青緑 (#4dd0e1) | ゆっくり膨張収縮 |
| **stress** | ダイヤモンド（◆） | 赤・紫 (#ff1744) | 震える（微細な揺れ） |
| **sadness** | 滴型（💧） | 紺色 (#1a237e) | ゆっくり揺れる |
| **inspiration** | 六角形（⬡） | 虹色（グラデーション） | 回転+パルス |
| **nostalgia** | 雲型（☁） | セピア (#8d6e63) | 浮遊（上下にゆっくり移動） |
| **confusion** | 歪んだ四角形（▱） | 濁った緑 (#66bb6a) | 不規則な動き |

### 2. サイズと視認性

- **基本サイズ**: 強度0.0で8px、強度1.0で16px
- **最小サイズ**: 6px（視認性を確保）
- **最大サイズ**: 24px（画面を圧迫しない）
- **影と光**: 各オブジェクトに発光効果（box-shadow）を追加

### 3. アニメーション

- **joy**: 回転アニメーション（2秒周期）+ パルス（1秒周期）
- **peace**: 膨張収縮（3秒周期）
- **stress**: 微細な揺れ（0.1秒周期、ランダム）
- **sadness**: ゆっくり揺れる（2秒周期）
- **inspiration**: 回転（3秒周期）+ パルス（1.5秒周期）
- **nostalgia**: 上下に浮遊（4秒周期）
- **confusion**: 不規則な動き（ランダム）

### 4. ドットライクな質感

- **エッジ**: 少しシャープなエッジ（border-radiusを控えめに）
- **ピクセル感**: 必要に応じてピクセルアート風のエッジ
- **色のコントラスト**: 背景とのコントラストを確保

## 🛠 実装案

### Option 1: CSSのみで実装（シンプル）
- CSSの`clip-path`や`transform`で形状を作成
- CSSアニメーションで動きを表現
- 軽量で高速

### Option 2: SVGアイコンを使用
- 各感情にSVGアイコンを用意
- より詳細な形状が可能
- ファイルサイズが増える可能性

### Option 3: Canvas APIで描画
- 完全にカスタマイズ可能
- ドットライクな質感を実現しやすい
- 実装が複雑

## 💡 推奨実装

**Option 1（CSSのみ）**を推奨します。理由：
- シンプルで保守しやすい
- パフォーマンスが良い
- 2Dドットライクな表示に十分
